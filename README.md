# 基于vue2.0+vuex开发的本地待办事项管理

> 本文采用vue2.0+vuex+webpack，实现一个简单的todolist列表。

## 功能说明
- 支持事件状态切换
  - 未完成 -> 已完成(勾选checkbox)
  - 已完成 -> 未完成(取消勾选checkbox)
- 支持编辑事件
- 支持添加事件
- 支持删除事件

## 安装步骤
本项目是使用vue-cli脚手架生成的项目，项目代码可以到我的github上clone下来。clone下来之后可进入文件目录

	git clone https://github.com/clara222/todoList.git
	cd todoList
	npm install
	npm run dev
	//安装成功后再看看是否可以正确运行了
